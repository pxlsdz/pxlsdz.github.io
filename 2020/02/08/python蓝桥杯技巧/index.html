队列与优先队列

1.线程队列Queue — FIFO(先进先出队列)，即哪个数据先存入，取数据的时候先取哪个数据，同生活中的排队买东西；

2.线程队列LifoQueue — LIFO(先进后出队列)，即哪个数据最后存入的，取数据的时候先取，同生活中手枪的弹夹，子弹最后放入的先打出；

3.线程队列PriorityQueue — PriorityQueue(优先级队列)，即存入数据时候加入一个优先级，取数据的时候优先级最高的取出；
