重定向

执行一个shell命令行时通常会自动打开三个标准文件：

  1. 标准输入文件（stdin）代码为 0 ，使用 < 或 << ；
    0代表：/dev/stdin
  2. 标准输出文件（stdout）代码为 1 ，使用 > 或 >> ；
    1代表：/dev/stdout
  3. 标准错误输出文件（stderr）代码为 2 ，使用 2> 或 2>> ；
    2代表：/dev/stderr

​ 进程将从标准输入文件中得到输入数据，将正常输出数据输出到标准输出文件，而将错误信息送到标准错误文件中。

/dev/null:特殊文件，无限大的空洞

/var/log:日志文件夹

管道

​ 管道可以把一系列命令连接起来，这意味着第一个命令的输出会作为第二个命令的输入通过管道传给第二个命令，第二个命令的输出又会作为第三个命令的输入，以此类推。

语法：command1|command2|command3|command4

举例：ls -la |more

1.cut
功能：将同一行的数据（以指定字符）进行分解。（一般用于分析日志文件）
语法：cut –d “分割字符” [-cf] fields
参数说明：
－d：后面接的是分隔字符，默认是空格符
－c：后面接的是第几个字符
－f：后面接的是第几个区块
举例：将PATH变量取出，要找到第三个路径

2.wc
功能：显示文件的大小，可以显示行数、单词数和字符数
语法：wc [-lwm]
参数说明：
-l:紧显示多少行
-w:紧显示多少字（英文单词）
-m:多少字符

举例：/etc/host.config里到底有多少相关字、行、字符数

【例】解析命令cat /etc/passwd | grep /bin/bash | wc –l

利用第一个管道将cat命令（显示passwd文件的内容）的输出送给grep命令，grep命令找出含有 “/bin/bash”的所有行；

第二个管道将grep的输入送给wc命令，wc命令统计出输入中的行数。这个命令的功能在于找出系统中有多少个用户使用 bash。

5.设置别名（alias）

  1. 设置别名

    设置别名的命令格式如下：

    alias [别名]=[需要定义别名的命令]

    例：以长格式的形式查看当前目录下的所有文件，包括隐藏文件，可以设置别名alias ll=‘ls –al’。

    注意：等号两端不能空格，字符串需要加引号。

  2. 查看别名

    查看当前用户的所有别名:$ alias

  3. 取消别名:

    $ unalias 命令

  4. 设置别名永久生效

    别名在每次登录时都有效，需将alias命令写到初始化脚本文件（.bashrc）中。

    举例：设置别名rm=’rm –i’，使用rm删除命令删除时每次都有提示确认信息，并让别名设置永久生效。

12.解释命令串的作用：grep /bin/bash /etc/passwd 2>/dev/null|sort –k 1 >/tmp/users.txt 2>/dev/null && cat /tmp/users.txt|wc –l

(体会&&的用法，用于连接多个命令的使用，前一个命令正确，才会执行&&后的命令)

  1. grep /bin/bash /etc/passwd 2>/dev/null
    grep /bin/bash /etc/passwd: 在 /etc/passwd文件中，寻找“/bin/bash”。

    2>/dev/nul:忽略错误出现。

  2. sort -k 1 >/tmp/users.txt 2>/dev/null

    sort -k 1将得步骤1的结果按第一列升序排序；

    /tmp/users.txt：得到的结果存入/tmp/users.txt中，

    2>/dev/null：运行错误忽略。

  3. &&为之前命令运行正确则执行。即如果之前命令出错，那么&&后面的不会执行。

  4. 如果正确cat /tmp/users. txt ：读取之前的结果文件/tmp/users. txt，

  5. wc -l：返回一共由多少行
