shell定义

Shell：字面意思“壳”；引申一下就是人机界面。是Linux里用户使用Linux的桥梁（命令接口）。

Bourne（[buən]） Again Shell (bash)：GNU计划的一部分，用来替代Bourne shell。用于基于GNU的系统.
大多数的Linux(Red Hat,Centos, Slackware, Caldera)都以bash作为缺省的shell

shell实用功能

快捷键

Tab键：自动补齐

反斜杠“\”：强制换行

快捷键 Ctrl+U：清空至行首

快捷键 Ctrl+K：清空至行尾

快捷键 Ctrl+L：清屏

快捷键 Ctrl+C：取消本次命令编辑

Home：跳到行首

End：跳到行尾

历史命令的使用

bash支持历史命令功能，通过上下光标键来选择，history命令可以查看执行的命令

Shell还可以执行指定序号的历史命令,使用方式!num

当历史命令比较多时，可以结合grep命令和管道进行查找使用

通配符 说明
？ 匹配任意一个字符
* 匹配任意多个字符
[] 相当于或的意思
- 代表一个范围，比如a-z

Linux目录类命令

常见目录

pwd 绝对路径

显示当前工作目录的绝对路径

命令格式：pwd

【 例】显示用户当前工作目录路径

[user@localhost ~]$ pwd

/home/user

注意：

  • 绝对路径：指从根目录（/）开始到当前目录（文件）的路径；
  • 相对路径：指从当前目录到其下子目录（文件）的路径。

mkdir 创建目录

命令格式： mkdir [选项] [目录]

选项 说明
-p 若路径中的目录不存在，先创建目录
-v 每次创建目录都显示信息

【例】 创建空目录test。

[user@localhost ~]$ mkdir test //当前目录下创建空目录test

【例】 递归的创建目录。

[user@localhost ~]$ mkdir -p test1/test2

cd 更改工作目录路径

命令格式： cd [目录]

几个特殊符号：

  1. . 代表当前所在目录
  2. .. 代表当前目录位置的上一层目录
  3. ~ 代表家目录（home directory),即login时所在的目录
  4. ~ user 代表到 user 用户的根目录

ls 列出目录和文件信息

命令格式： ls [选项] [目录] [文件]

选项 说明
-l 以详细信息的形式展示出当前目录的文件
-a 显示当前目录中的全部文件（包括隐藏文件）
-d 查看目录属性
-t 按照创建时间顺序列出文件
-F 显示文件类型

Linux中一切皆文件
常见文件：
1）普通文件 文件属性的第一个符号是“-”
2）目录文件 文件属性的第一个符号是“d”
3）设备文件 /dev中有大量设备文件，包括块设备文件（b）和 字符设备文件（c）
4）管道文件 从一头流入从另一头留出的文件，文件属性的第一个符号是“p”
5）链接文件 文件属性的第一个符号是“l”

文件权限

举例1：$[-][r w x][r - x][r - -]$
举例2：$[d][r w x][r - -][r - -]$
问题：非root的其他人是否可以进入该目录？
注意：在Linux中，文件是否能执行，通过是否具有x属性来决定执行，与文件名没有关系；x与目录的关系相当重要，如果在该目录下不能执行命令，也就无法进入

小结

要完成的操作 命令
返回当前用户的主目录 cd
确定当前位置的绝对路径名 pwd
返回到最前面的工作目录 cd ~
从当前位置上移两个级别 cd ../..
列出当前的位置（长格式）及隐藏文件 ls -la
从任何当前位置移动到二进制文件（*/bin）位置 cd /bin
上移到当前位置的父目录 cd ..
从根目录移到二进制文件（/bin）位置 cd bin
当前用户的主目录不是当前位置时，以最简单的语法列出该目录（长格式） ls -l ~

rmdir 删除空目录

命令格式： rmdir [选项] [目录]

选项 说明
-p 删除递归目录，当子目录删除后其父目录为空时，也一同被删除
-v 输出处理的目录详情

文件操作命令

touch 创建空文件、更改文件时间

命令格式： touch [选项] 文件

选项 说明
-a 只更改访问时间（atime）
-m 更改文件的修改时间记录（mtime）
-c 假如目标文件不存在，则不会建立新的文件
-d<字符串> 使用指定字符串表示时间而非当前时间
-t<日期时间> 使用MMDDhhmm格式的时间而非当前时间

时间格式MMDDHHmm指月（MM）日（DD）时（HH）分（mm），如果还需要加上年份，可以使用201903121930，表示2019年3月12日19时30分。

cp 复制文件和目录

命令格式： cp [选项] [源文件|目录] [目标文件|目录]

选项 说明
-f 强制复制文件和目录，无论目的文件或目录是否已经存在
-r 递归复制目录下的子目录和文件
-i 若目标文件存在，则提示用户如何操作

mv 文件和目录改名、移动文件和目录路径

命令格式：mv [选项] [源文件|目录] [目标文件|目录]

选项 说明
-f 移动时自动替换已经存在的目标文件，不提示
-i 若目标文件存在，则提示用户如何操作

find 用于查找文件或目录

命令格式：find [查找范围] [选项] [查找条件]

如果用户没有指定查找范围，则find命令从当前目录开始搜索查找

选项 说明
-name<文件名*> 匹配文件的名称
-user <用户名> 匹配文件的所有者
-perm <权限> 匹配文件的权限
-type <文件类型> 匹配文件类型查找
-size n[ckMG] 匹配文件的大小（+50k查找超过50k的文件,而-50k则代表查找小于50k的文件）
—exec {} \; 后面可接对搜索到结果进一步处理的命令
